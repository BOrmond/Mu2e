Mu2e Tracker Database



This database houses all testing data for Mu2e tracker components:

DMB
  • AMB (Cal)
  • AMB (HV)
  • Preamp

    • To add a component to the database, select the component above and insert all available information. Crosstalk and continuity files should be submitted in the order that component pins were tested (multiple files may be added at once). Enter a component index before creating the entry.

    To find data for a particular component, select "Search" in the navigation bar and enter the component index (found on the label). Components may be edited or deleted from the search page.

    This database was made using Python-Flask, Jinja2, Bootstrap, and SQLAlchemy for the LBNL branch of Mu2e
